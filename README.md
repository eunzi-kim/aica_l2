# 🤖 AICA L2 합격 챗봇

**AICA L2 시험 대비용 AI 기반 퀴즈 챗봇**

## ✨ 주요 기능

- 📚 **200개의 AI/윤리 관련 문제** - AICA L2 시험 출제 기준 준수
- 📱 **완벽한 모바일 반응형** - 핸드폰, 태블릿, 데스크톱 모두 최적화
- 📊 **실시간 학습 대시보드** - 정답률, 오답 통계 제공
- 💾 **로컬 학습 기록 저장** - 앱을 껐다 켜도 진행도 유지
- ❌ **오답 노트 모드** - 틀린 문제만 모아서 복습
- 🎨 **모던 UI/UX** - 프리미엄한 사용자 경험

## 🚀 온라인 사용 (배포됨)

[🌐 여기를 클릭해서 온라인으로 사용하기](https://aica-l2-quiz.streamlit.app)

## 💻 로컬 설치 & 실행

### 1. 파이썬 설치
- Python 3.8 이상 필요

### 2. 저장소 클론
```bash
git clone <저장소_주소>
cd quiz
```

### 3. 패키지 설치
```bash
pip install -r requirements.txt
```

### 4. 앱 실행
```bash
streamlit run app.py
```

**✨ 브라우저에서 자동으로 앱이 열립니다!**


## 🎯 사용 방법

### 1. 학습 모드 선택
- **전체 문제 풀기**: 처음부터 200개 문제 학습
- **오답 노트 풀기**: 틀린 문제만 복습

### 2. 문제 풀기
- 질문을 읽고 4개 선택지 중 하나 선택
- "정답 제출하기" 버튼 클릭
- 즉시 정답 여부 & 해설 제공

### 3. 학습 기록 추적
- 실시간 점수 표시
- 오답 노트 자동 저장
- 화면 종료 후에도 기록 유지

### 4. 초기화
- "학습 기록 초기화" - 모든 기록 삭제 후 새로 시작
